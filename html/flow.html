Canteen Ordering System

Project Flow + Techs Overview

1. Student Flow
1. Open app → View canteens
2. Choose canteen → View menu
3. Add items to cart
4. Checkout → Pickup slot → Payment (UPI/Cash)
5. Track order: Placed → Accepted → Preparing → Ready → Picked
2. Shopkeeper Flow
• See incoming orders (via 5s polling)
• Accept or Reject order
• Mark Preparing after accepting
• Mark Ready when food is done
• Mark Picked when student collects
• Set items In Stock / Sold Out

4. Backend Responsibilities
• Build Express server + folder structure
• Create APIs for Login / Signup (JWT Auth)
• Menu APIs: Add, Update, Delete, Toggle Availability
• Order APIs: Place Order, Status Update, Fetch Orders
• Payment Integration (Razorpay/UPI simulation)
• Connect frontend to backend using Fetch API
• Handle validation + middleware
• Host backend on Railway
